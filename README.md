# Personal Blog Website

A simple, dynamic, and responsive personal blog website built with **HTML**, **CSS**, and **JavaScript**. This blog shares thoughts, ideas, and projects, focusing on learning, growth, and web development.

## Technologies Used

- **HTML**: For building the structure of the website.
- **CSS**: For styling and layout.
- **JavaScript**: For interactivity (e.g., dark mode, form validation).
- **Parcel**: For bundling the assets and building the app.

## Features

- **Responsive Design**: The website is mobile-friendly and adapts to different screen sizes.
- **Dark Mode**: Toggle dark mode to enhance readability in low-light environments.
- **404 Page**: A custom 404 error page that guides users back to the homepage.
- **Sitemap**: A simple sitemap to help users navigate the website easily.
